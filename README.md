# Student Grade Management System

## Project Purpose

This project is a Java console-based application developed to manage student academic records digitally. The system allows teachers to add student details, display all students, search students by ID, and calculate average marks.

## Features

* Add a student
* Display all students
* Search student by ID
* Calculate average marks

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* GitHub Version Control

## How to Run the Program

1. Open the project in a Java IDE such as IntelliJ IDEA, Eclipse, or NetBeans.
2. Compile the Java files.
3. Run the `StudentGradeSystem.java` file.
4. Use the menu to manage student records.
